Guide d'utilisation

Bienvenue sur la plateforme de QCM d'ALTRAN,

Lisez attentivement ce guide d'utilisation avant d'utiliser notre plateforme de QCM.

Les tests que vous passerez seront des tests multipages comportant une à plusieurs questions par page.
Pour pouvoir naviguer à travers le test vous disposez de deux liens suivant et précédent vous menant respectivement vers la page suivante et la page précédente. Il y a aussi une liste de numéros de questions que voulez répondre. Le bouton vert indique que cette question est répondue.

Aprés avoir répondu à toutes les questions vous pourrez valider votre réponse via le bouton d'envoi.

  • Valider votre envoie terminera le test.Vérifiez que vous avez bien répondu à toutes les questions avant de valider vos réponses.
  • Les tests sont chronométrés.A la fin du temps imparti le test sera terminé et vos réponses seront sauvegardées.
  • Toute tentative de fraude est détectée. Si vous consultez des ressources externes d'informations plus qu'une fois, le test sera arrêtée.
  • Certains tests ne peuvent être passé qu'une seule fois. Après le début d'un test ne rafraichissez jamais la page et ne quittez pas le test sans valider .Sinon votre session sera fermée automatiquement sans validation de vos réponses et vous n'aurez pas la possibilité de repasser le test sans l'intervention d'un administrateur d'ALTRAN.

Quand vous êtes authentifiés, les tests qui vous sont affecté sont sur la colonne de gauche de l'écran. Il suffit de cliquer sur "QCM" pour consulter la liste complète des quizzes que vous devez passer.


Bonne chance !
